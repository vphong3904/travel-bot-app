1. Đầu tiên tạo file docker-compose.yml thêm bên dưới
services:
  postgres:
    image: pgvector/pgvector:pg17
    container_name: travel_postgres
    restart: unless-stopped
    environment:
      POSTGRES_DB: pdtrip_ai_db
      POSTGRES_USER: user
      POSTGRES_PASSWORD: 12345678
    ports:
      - "5432:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data
      - ./initdb:/docker-entrypoint-initdb.d

volumes:
  postgres_data:

2. Sau đó tạo folder nằm trong backend tên initdb rồi quăng file pdtrip_ai_db.sql vào
Kết quả sẽ có đường dẫn: initdb\pdtrip_ai_db.sql

3. Tạo container trong docker
docker compose up -d

4. Kiểm tra container trên đã tạo chưa
docker ps
Kết quả:
travel_postgres
pgvector/pgvector:pg17
5. Kiểm tra lại log ra coi có tạo chưa:
docker logs -f travel_postgres
Kết quả:
CREATE EXTENSION
CREATE FUNCTION
CREATE TABLE
CREATE INDEX
INSERT 0 ...
PostgreSQL init process complete
database system is ready to accept connections
Sau tạo Ctrl + C thoát log
6. Nhập ở file sql
POSTGRES_DB: pdtrip_ai_db
POSTGRES_USER: postgres
7. chạy lệnh vào mở PostgreSQL
docker exec -it travel_postgres psql -U postgres -d pdtrip_ai_db
8. Nhập \dt để kiểm tra tất cả bảng có trong db muốn thoát nhập \q vừa nhập là nó thoát liền luôn 
9. Sau khi thoát ở trên có thể hoặc ko nhập để xem coi data có add đủ dòng không kết quả sẽ là số dòng của bảng đó
SELECT COUNT(*) FROM destinations;
SELECT COUNT(*) FROM hotels;
SELECT COUNT(*) FROM tours;
SELECT COUNT(*) FROM users;
10. Nếu muốn thoát tất cả nhập \q lần nữa r enter là thoát luôn khỏi PostgreSQL về với path backend
